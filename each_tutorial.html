Way too long words: link to the problem

In this problem, you're asked to shorten words that have more than 10 characters like this: "information" should be shortened down to "i9n", which means that there are 9 letters in between "i" and "n" excluding the two. Words that have 10 characters and below should be left alone, so, "hello" should remain "hello" in the output.

The first line of the input is a number that is the number that is the number of iterations the input should be asked. For example, if the first line of the input is 2, then you will be asked for a word two times.

Then, the user should be asked for a word for the specified number of times. After every iteration, the output must be the word, either shortened (if the word has more than 10 characters) or not shortened (if the word has 10 or less number of characters).

t = int(input())
for i in range(t):
 word = input()
 if len(word) > 10:
  print(word[0]+str(len(word) - 2)+word[-1])
 else:
  print(word)

Team: link to the problem

Here, there are three friends who decide to take part in a competetive programming competition. These three, Petya, Vasya, and Tonya, form a team. They decide that, if less than two of them knows the answer to the given problem, then they will not attempt it. If two or more of the three know how to solve the problem, they would solve it.

The first line of the input is the number of questions there are going to be in the competition. If the number of questions is 3, then the user will be asked for input three times.

The next line(s) are the statements. The syntax of the statements is as follows:

The first number in the statement will be Petya's: if Petya is sure she knows the answer, this will be 1. If not, it'll be 0.

The second number in the statement is be Vasya's: if she's sure she knows the answer, this will be 1. If not, it'll be 0.

The third number in the statement will be Tonya's: if she is sure she knows the answer, this will be 1. If not, it'll be 0.

The output should be the number of questions in the competition that they will solve.

count = 0
for i in range(int(input())):
 statement = input()
 l = statement.split(' ')
 total = 0
 for j in l:
  total += int(j)
 if total >= 2:
  count += 1
print(count)
